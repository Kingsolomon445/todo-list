Getting Started

To use the API, you will need to obtain an authentication token by making a request to the endpoint 

/api/api-token-auth/
. You will need to pass your username and password in the request body in json format. The obtained token should then be included in the header of each subsequent request as Authorization: Token token.

API Documentation

This page will provide a detailed explanation of the API endpoints for the superToDo app.

Endpoints

  • Add a Task

    To add a task, make a POST request to the following endpoint:

    /api/tasks/

    In the request body, include the following data:

    {
    "title": "Task Title",
    "description": "Task Description",
    "due_date": "YYYY-MM-DD"
}

    This will add the task with the given title and description to your list of tasks.

  • Get All Tasks

    To retrieve all tasks, make a GET request to the following endpoint:

    /api/tasks/

    This will return a JSON list of all tasks and their respective data.

  • Get a Task by ID

    To retrieve a specific task by ID, make a GET request to the following endpoint:

    /api/tasks/<id>/

    Replace <id> with the ID of the task you wish to retrieve. This will return a JSON object of the task data.

  • Update a Task

    To update a task, make a PUT request to the following endpoint:

    /api/tasks/<id>/

    Replace <id> with the ID of the task you wish to update. In the request body, include the updated task data in the same format as when adding a task.

  • Delete a Task

    To delete a task, make a DELETE request to the following endpoint:

    /api/tasks/<id>/

    Replace <id> with the ID of the task you wish to delete. This will remove the task from your list of tasks.
